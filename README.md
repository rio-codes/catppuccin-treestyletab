## Usage

1. Clone this repository locally
2. Open Tree Style Tab preferences
3. Under "Appearance", select "No decoration"
4. Under "Advanced" > "Extra style rules for content provided by Tree Style Tab", select "Load from File"
5. Choose your preferred css file from the cloned repository
